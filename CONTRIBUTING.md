# Contributing to Gemini Test

## Git Flow
1. Crie uma branch a partir da `main` (`feature/nome-da-feature`).
2. Faça seus commits seguindo o padrão [Conventional Commits](https://www.conventionalcommits.org/).
3. Abra um Pull Request para a `main`.

## Coding Standards
- Use componentes funcionais em React.
- Mantenha o código limpo e comentado quando necessário.
